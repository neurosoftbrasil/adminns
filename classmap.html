ADMINNS

  • Banco de dados: MySql - UTF8-bin
  • App // Pasta de aplicação
    • App.php // Instanciada | Cria ambiente
    • Config.php // Estatica | Configura variáveis da aplicação
    • Controller
      • HelpController.php // Instanciada | Controller de funções de ajuda simples.
      • HomeController.php // Instanciada | Controller da página Home
      • LoginController.php // Instanciada | Controller das páginas de Login
      • PerfilController.php // Instanciada | Controller da página de perfil
      • SecureController.php // Instanciada | Controller seguro extendido do controller Library/Controller.php. Valida se há usuário logado.
      • SuporteController.php // Instanciada | Controller do módulo de suporte
      • UsuarioController.php // Instanciada | Controller do módulo de controle de usuários
  • Library // Pasta de framework
    • Db.php // Instanciada | Conecta ao banco pela configuração das variáveis de aplicação
    • Helper.php // Estatica | funções de transformação de dados de Views
    • Util.php // Estatica | funções de transformação de dados para Debug
    • Session.php // Estatica | Funções de autenticação e autorização
    • Request.php // Estatica | Funções de manipulação de url
    • Router.php // Instanciada | Funções de redirecionamento de páginas via REST
    • FormHelper.php // Estatica | Helper de impressão de formulários
  • Layout // Pasta de layout e temas
    • header.php // Include | Cabeçalho da template.
    • footer.php // Include | Rodapé da template.
    • fonts // Anexo | Fontes do Bootstrap Framework.
    • images // Anexo | Diretório de imagens.
    • javascript // Anexo | Arquivos-fonte do Bootstrap, jQuery, jQuery UI e demais scripts JS.
    • stylesheets // Anexo | Folhas de estilho do Bootstrap e do sistema
    • Usuario // Pasta do controller UsuarioController
      • index.tpl // Include | View do método UsuarioController->index()
      • editar.tpl // Include | View do método UsuarioController->editar()
    • ...
  • index.php // Configura variáveis do framework
